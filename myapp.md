App Name:
TickFair Connect

1.1 One-sentence pitch:
A mobile app that allows users to join a fair queue and reserve one
ticket for small event sales using a simple and controlled system.

1.2 App category:
Entertainment

2.1 Persona:
Age: 22
Occupation: University Student
Location: Thailand
Tech Skill Level: Intermediate
Daily Problems: Misses tickets, unclear queue order

2.2 Daily Problem:
She often cannot buy tickets because many people enter at the same
time and there is no clear or fair queue order.

3.1 Pain Point:
• Many users access at the same time
• No visible queue order
• Users can try repeatedly
• No simple reservation step

3.2 Why existing solutions not enough:
• No simple queue system
• No ticket limit per user
• No clear reservation step

4.1 What the app does:
TickFair Connect lets users log in, select an event, join a queue, and
reserve one ticket when their turn arrives.

4.2 How it solves the problem:
• Uses first-come-first-served queue
• Allows only one ticket per account
• Limits number of users per event

5. Core Features (MVP):
• Register / Login
• Event List
• Join Queue
• Queue Position Display
• Reserve Ticket

6. User Journey:
1. User registers
2. Logs in
3. Selects event
4. Joins queue
5. Sees queue position
6. Waits for turn
7. Reserves ticket
8. Sees confirmation
